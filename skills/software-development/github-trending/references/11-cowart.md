## Summary
Local infinite canvas plugin for Codex, built on tldraw. Visual canvas for ideation, annotation, image generation, and iterative image editing. Data saved to project's `canvas/` directory.

## What to take
- **tldraw as visual layer**: open-source infinite canvas library for web
- **Local-first data**: canvas data saved to project directory, not cloud
- **Agent + canvas integration**: agent reads/writes canvas state for visual workflows
